# Demo steps

1. Start the app.
   - Mac/Linux: `./run-demo.sh`
   - Or run backend and frontend separately.
2. Open `http://localhost:3000`
3. Sign in as `mbma2264@gmail.com`
4. Click **Load demo sample data**
5. Show the dashboard cards, saved records, and activity feed
6. Use **Try the classifier** to save a manual example
7. Optionally click **Connect Gmail** to show real inbox sync

## What to say in the demo
- The app explains *why* a message is risky, not just a label.
- Manual classifications and Gmail imports both persist to the user-scoped record store.
- Activity is logged, so the user can see what happened and when.
- Gmail connection is optional for the first minute of the demo because sample records make the dashboard feel alive immediately.
